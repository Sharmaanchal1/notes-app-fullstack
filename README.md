# Notes App (Full Stack)

A full-stack notes application built with:

- React (Frontend)
- Node.js + Express (Backend)
- MySQL (Database)
- JWT Authentication

## Features
- User authentication (login/signup)
- Create, edit, delete notes
- Pin notes
- Search notes
- Undo delete
- Responsive UI

## How to Run
1. Clone the repo
2. Install dependencies in frontend and backend
3. Add .env file
4. Run backend and frontend servers
